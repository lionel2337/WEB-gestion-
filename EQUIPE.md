## 🛠️ Boîte à Outils & Technologies Utilisées

Afin de bâtir une application performante, robuste, moderne et sécurisée, les outils et technologies suivants ont été adoptés :

### 🔹 1. Technologies Cores & Langages
* **PHP 8+** : Utilisé pour toute la logique backend sous forme de classes Orientées Objet (POO), garantissant la modularité et l'évolutivité du code.
* **HTML5 & CSS3** : Pour une structure de page sémantique et un style visuel haut de gamme et responsive.
* **JavaScript (Fetch API / AJAX)** : Permet d'effectuer des requêtes asynchrones en arrière-plan sans recharger la page (moteur de recherche en temps réel, pointage dynamique, formulaires réactifs).

### 🔹 2. Gestion & Persistance des Données
* **MySQL** : Système de Gestion de Base de Données (SGBD) relationnelle assurant le stockage sécurisé des 34 tables et 5 vues.
* **PDO (PHP Data Objects)** : API de connexion sécurisée avec requêtes préparées systématiques, bloquant nativement les injections SQL.
* **XML & XSD Schema** : Format d'échange et de sauvegarde des données employé pour l'export des fiches des employés, validé rigoureusement par un schéma de validation XSD personnalisé.

### 🔹 3. Environnement de Développement & Collaboration
* **XAMPP (Apache + MySQL)** : Serveur local de développement utilisé pour héberger l'application et la base de données de test en local.
* **phpMyAdmin** : Interface d'administration web pour la gestion, la structure et le débogage visuel des tables SQL.
* **Git & GitHub** : Outil de contrôle de version distribué permettant le travail collaboratif en équipe sans perte de code ni conflits d'écrasement.

### 🔹 4. Bibliothèques Graphiques & Expérience Utilisateur (UI/UX)
* **Bootstrap 5.3** : Framework CSS utilisé pour la grille responsive et les structures de mise en page de base.
* **FontAwesome 6** : Collection d'icônes vectorielles modernes utilisées dans le menu latéral et toutes les actions interactives.
* **Chart.js 4** : Bibliothèque de graphiques dynamiques HTML5 utilisée sur le Tableau de bord pour les analyses statistiques en temps réel.
* **Google Fonts (Inter & Poppins)** : Typographies premium pour une lisibilité parfaite et un design élégant rappelant les grandes applications SaaS.
* **SweetAlert2 & Animate.css** : Utilisés pour fournir des fenêtres de dialogue interactives, des alertes de confirmation soignées et des micro-animations de transition haut de gamme.
